# Compilador t++

- Command line:
	- python3 lexer.py exemplo.tpp

### Como utilizar o parser.py
- Command line:
	- python3 parser.py exemplo.tpp

### Como utilizar o semantica.py
- Command line:
	- python3 semantica.py exemplo.tpp

### Como utilizar o gen.py
- Command line:
	- python3 gen.py exemplo.tpp
